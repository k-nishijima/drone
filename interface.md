# Arduino <-> Raspberry Pi シリアルインターフェイス仕様

### ラズパイを親、Arduinoを子とする。

### コマンドの末尾は「|」とする。

## 移動に関する仕様

親→子へのリクエスト：

* front:nnnn
* back:nnnn
* stop
* left:nnnn
* right:nnnn

nnnn = 数字でミリ秒を指定

例： forward:5000 = 5千ミリ秒前進



## GPSに関する仕様

親→子へのリクエスト：

* gps

（2秒待って）
レスポンス：GPSから返ってきた生データを1行返す

データ例：$GPGGA,085420.000,2628.5327,N,12750.0444,E,1,03,4.6,38.9,M,29.3,M,,0000*64


* 北緯：2628.5327
* 東経：12750.0444
